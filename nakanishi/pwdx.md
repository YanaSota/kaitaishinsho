# pwdx
プロセスの作業ディレクトリを表示するコマンド

  実行例 [](変更しない)
  
  ```
  pwdx 765856
  ```


  実行結果　[](変更しない)


  ```
  $ps  //プロセスIDの確認
      PID TTY          TIME CMD
 765856 pts/0    00:00:00 bash
 766065 pts/0    00:00:00 ps

  $ pwdx 765856 
  765856: /home/user/pwdx //PIDが765856の作業ディレクトリを表示する
  ```
