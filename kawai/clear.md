# clear
入力や出力をクリアする。画面をクリアする。わかりやすさのため、入力だけでなくユーザ名まで実行例に示す。

  実行例 [](変更しない)
  
  ```
  usr@usr1:~$ aa
  コマンド 'aa' が見つかりません。次の方法でインストールできます:
  sudo apt install astronomical-almanac
  usr@usr1:~$ clear
  ```


  実行結果　[](変更しない)


  ```
  usr@usr1:~$
  ```

### オプション一覧


- **-V**
  
  このプログラムで使用されたncursesのバージョンを報告し、終了する。画面はクリアされない。

  実行例 [](変更しない)
  
  ```
  clear -V
  ```


  実行結果　[](変更しない)


  ```
  ncurses 6.3.20211021
  ```
