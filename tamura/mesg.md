# mesg
`mesg` コマンドは、システム上の他のユーザーが `write` コマンドまたは `talk` コマンドを使用して、 メッセージを送信できるかどうかを制御するコマンド。引数なしで呼び出されると、`mesg` コマンドは、ターミナルの現在のメッセージ許可設定を表示する。
root 権限を持つユーザーは、メッセージ許可設定に関係なく、どのターミナルにでもメッセージを送信できる。

次に示すフラグによって制御する。

- n : メッセージを許可しない。
- y : メッセージの表示を許可する。

実行例 []()
```
mesg
```

実行結果 []()
```
is y
```
