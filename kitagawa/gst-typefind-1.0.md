# gst-typefind-1.0
gst → GStreamer  
GStreamerは、マルチメディアデータの処理やストリーミングを行うためのフレームワークであり、様々なメディア関連のタスクに使用される。  
gst-typefind-1.0 を実行すると、ファイルのメディアタイプを表示することができる。

  実行例 [](変更しない)
  
  ```
  $ gst-typefind-1.0 music.mp3
  ```


  実行結果　[](変更しない)


  ```
  music.mp3 - application/x-id3
  ```
