# < H1~6 > < /H1~6 >
HタグはHeading(見出し)の意味である。表示結果からもわかるようにHタグは< /H>で閉じなくともHタグが先頭にある行は一つの見出しになる。
構造と表記を意味するタグ。

記述例 [](変更しない)

```
<BODY>
    <H1>H1_H1</H1>
    <H2>H2_H2
    <H3>H3_H3
    <H4>H4_H4
    <H5>H5_H5</H5>
    <H6>H6_H6</H6>
</BODY>
```
